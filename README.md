# clash-vpn-lab

Личный исследовательский репозиторий конфигураций **Clash / Clash Meta / sing-box**  
с автообновляемыми VPN-провайдерами.

## 🎯 Цели проекта
- стабильные маршруты для:
  - 🌍 интернет-сёрфинга
  - 🤖 ChatGPT / OpenAI
  - 🎮 онлайн-игр (Roblox и др.)
- автообновляемые подписки
- разделение долгоживущих и краткоживущих серверов
- минимальный мусор

## 📁 Структура
- `providers/` — источники VPN (http subscriptions)
- `configs/` — готовые YAML для Clash
- `rules/` — правила маршрутизации
- `experimental/` — тесты и временные схемы

## 🔄 Автообновление
Все `proxy-providers` используют:
- `interval: 3600`
- `health-check: enable`

Clash обновляет конфигурации автоматически.
